	(then open xammp and start Apache and my SQL)


2.Then copy dynamic page folder and static page to localdisk C -> xammp -> htdocs 
3.then open any browser
4.then type localhost/(foldername)
5.then open the folder in browser and open dynamic pages and open CNA.html
